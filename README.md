# React useState App

A simple, interactive React application designed to demonstrate the fundamental concepts of state management using the `useState` hook. This project consists of two single-page components: a Counter and a Random Number Generator.

## 🚀 Features

### 1. Interactive Counter
A robust counting application that allows users to adjust a numeric value.
* **Increase:** Adds 1 to the current count.
* **Decrease:** Subtracts 1 from the current count (includes logic to prevent the counter from dropping below zero).
* **Reset:** Instantly resets the counter back to 0.
* **Conditional Rendering:** Displays a warning message if the user attempts to decrease the value below the minimum limit.

### 2. Random Number Generator
A dynamic component that generates numbers on the fly.
* **Generate:** Clicking the button calculates and displays a random number between 1 and 100.
* **Dynamic UI:** The UI instantly updates to display the newly generated number using React state.
